# dotfiles

## for MacOS

```
$ xcode-select --install
$ hash -r
$ git clone http://github.com/tk0miya/dotfiles $HOME/.dotfiles
$ cd $HOME/.dotfiles
$ chsh -s /bin/zsh
$ exec zsh
$ sh setup.sh
$ exec zsh
$ sh setup.sh
```

### Manually settings

iTerm2
------

In `[Preferences] > [General]`, put `/path/to/.iTerm2` to custom folder for preferences
